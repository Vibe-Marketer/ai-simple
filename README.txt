CREATIVE – DIGITAL AGENCY WEBSITE TEMPLATE

Thank you for purchasing Creative.

Creative is a high-impact, dark brutalist website template designed for digital agencies, creative studios, and designers who want to showcase work, services, and pricing with clarity and confidence.

Built to highlight real projects, explain services clearly, and convert visitors into inquiries and booked calls, Creative focuses on strong typography, structured layouts, and a modern agency aesthetic.

--------------------------------
HOW TO USE
--------------------------------

You have two ways to use this template:

Option 1 – Remix in Aura (Recommended)
- After purchasing, you can remix this template directly in Aura.
- This allows you to visually edit layouts, content, and styles
  without touching raw code.
- Ideal for fast customization and iteration.

Option 2 – HTML Editing
1. Open the HTML files in your browser to preview the template.
2. Edit files using a code editor (VS Code recommended).
3. Replace text, images, and branding with your own content.
4. Deploy to your hosting provider or integrate into your project.
