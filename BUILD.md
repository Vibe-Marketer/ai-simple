# AI Simple Website — Build Guide

## Setup
```
npm install
```

## Development
```
npm run dev   # Watch & recompile Tailwind CSS
```

## Production Build
```
npm run build  # Compiles and minifies Tailwind CSS to public/css/tailwind.css
```

## Image Optimization
```
npm run optimize-images  # Creates WebP versions of all images
```
Requires `sharp`: already in devDependencies.

## Deployment
Deploys automatically via Vercel on push to main. Build command: `npm install && npm run build:css`

## Switching from CDN Tailwind to Compiled CSS
After running `npm run build`, update each HTML page:
- Remove: `<script src="https://cdn.tailwindcss.com"></script>`
- Add: `<link rel="stylesheet" href="/css/tailwind.css">`

Note: After switching, any dynamic Tailwind classes (classes generated by JavaScript) will NOT be available unless they are included in the content scanning paths in tailwind.config.js.

## Environment Variables (Vercel)
- `SUPABASE_URL` — Supabase project URL
- `SUPABASE_SECRET_KEY` — Supabase service role key
- `STRIPE_SECRET_KEY` — Stripe secret key
- `STRIPE_WEBHOOK_SECRET` — Stripe webhook signing secret
- `COMPOSIO_API_KEY` — Composio API key
- `COMPOSIO_STRIPE_AUTH_CONFIG_ID` — Composio Stripe auth config ID (was hardcoded)
- `COMPOSIO_GOOGLEDRIVE_AUTH_CONFIG_ID` — Composio Google Drive auth config ID
- `COMPOSIO_RESEND_AUTH_CONFIG_ID` — Composio Resend auth config ID
- `COMPOSIO_WHATSAPP_AUTH_CONFIG_ID` — Composio WhatsApp auth config ID
- `COMPOSIO_GOOGLEDRIVE_USER_ID` — Composio Google Drive user ID
- `COMPOSIO_RESEND_USER_ID` — Composio Resend user ID
- `COMPOSIO_WHATSAPP_USER_ID` — Composio WhatsApp user ID
- `COMPOSIO_STRIPE_USER_ID` — Composio Stripe user ID
- `SETUP_SECRET` — One-time setup endpoint auth secret
- `GTM_CONTAINER_ID` — Google Tag Manager container ID (replace GTM-XXXXXXX in HTML)
